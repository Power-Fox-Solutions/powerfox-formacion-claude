# Weather Report

Aplicación de pronóstico meteorológico construida con React y Vite.

## Características

- Búsqueda de clima por ciudad
- Información detallada del clima actual (temperatura, humedad, viento)
- Pronóstico de 7 días
- Datos mock para Madrid, Barcelona y Buenos Aires

## Estructura

```
src/
  ├── components/
  │   ├── SearchBar.jsx
  │   ├── WeatherCard.jsx
  │   └── ForecastList.jsx
  ├── api/
  │   └── weather.js
  ├── App.jsx
  ├── App.css
  └── main.jsx
```

## Instalación y uso

```bash
npm install
npm run dev
```

## Ciudades disponibles

- Madrid
- Barcelona
- Buenos Aires
